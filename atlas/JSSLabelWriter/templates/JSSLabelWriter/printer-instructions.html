Zebra GX420t Setup Instructions

  1. Open the following web address: http://localhost:631/admin. If you get an error about the web interface being disabled, follow the instructions on the page to enable it.
  2. In the printers section click the "Add Printer" Button. If you recieve a login prompt use your local administrator credentials.
  3. From the list, select "AppSocket/HP JetDirect" and click continue
  4. In the connection field, enter the following: socket://10.21.99.44:9100
  5. Fill out the name, description and location fields. These fields are for your reference and can be anything
  6. From the list of Manufacters, select Zebra. For Model select Zebra ZPL Label Printer (en), then click "Add Printer".
  7. For Media Size select 3.00x2.00", ensure the resolution is set to 203dpi, the click "Set Default Options"
  8. The printer is now setup, please make sure that you select the correct media size in the browser and that margins is set to none
